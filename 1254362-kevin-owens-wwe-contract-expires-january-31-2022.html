Report: Kevin Owens WWE Contract Expires On January 31, 2022

Publish date: 2024-08-03

The exact date that ’ WWE contract expires has been revealed.

Last week, Fightful Select reported Kevin Owens’ WWE contract is set to expire in January 2022, which was much earlier than it was believed to run out. Now, both the Wrestling Observer Newsletter and Fightful Select have confirmed Owens’ deal runs out on January 31, 2022.

The specific date is notable because it would indicate that if he does not re-sign with the company, Owens is also in jeopardy of missing the Royal Rumble event. The Rumble is typically an event held in January, but Fightful Select also recently reported that WWE is actually targeting early February as a possible date for the PPV. (It’s worth noting that WWE has already confirmed its “Day 1” pay-per-view event in Atlanta will take place on New Year’s Day, so there will still be a January event no matter when the Rumble takes place.) Additionally, Fightful also reported St. Louis is considered to be a front-runner to host the Royal Rumble, but plans are not set in stone for the host city yet.

Owens took part in a 2018 interview and said that he’d signed a new five-year contract, which would have taken him through 2023. Fightful’s Sean Ross Sapp explained that WWE restructured multiple contracts before the pandemic, and the end date of Owens’ deal was affected. It’s unknown why this change was made, but Sapp once again clarified that the deals were renegotiated ahead of the pandemic, not at the start of it.
